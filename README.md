# Smart Skincare Advisor

AI-powered skincare product analyzer that helps users 
find out if a product is suitable for their skin type.

## Current Status
Under Development

## Features Completed
- Skin type selection
- Product image upload
- OCR-based ingredient extraction
- Ingredient compatibility analysis
- Suitable / Caution / Avoid recommendation

## Features Coming Soon
- PostgreSQL database integration
- CNN product classifier
- AWS deployment
- Premium UI

## Tech Stack
- Streamlit, EasyOCR, Python, PostgreSQL, AWS
